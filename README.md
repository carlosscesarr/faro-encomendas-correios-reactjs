## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/carlosscesarr/faro-encomendas-correios-reactjs.git && cd faro-encomendas-correios-reactjs
```

Follow the steps below
```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn start
```
